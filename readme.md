## 定期試験

### 後期中間試験

手書きレポート，電卓は持ち込み可能である．留学生のみ電子辞書を持ち込んでも良い．テスト範囲は第7回までに取り扱った内容．

### 後期期末試験

テスト範囲は第8回から14回に取り扱った内容（資料７から１３までの内容の内，講義で扱ったもの）．

### 第1回レポート課題

第7回までの講義内容をまとめる．11月30日の講義終了後に提出する．

### 第2回レポート課題
